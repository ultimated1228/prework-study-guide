CSS


    Notes

  • CSS contains the style rules for the webpage
  • There are 3 types of CSS files: inline CSS, internal style sheets, and the most common external CSS style sheet
  • Declarations contain a property and a value for that property
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
